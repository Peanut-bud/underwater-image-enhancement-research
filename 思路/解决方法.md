# 面向人工光照水下图像的综合增强方法

你的图片核心问题是：

```text
黄绿色偏色
整体浑浊、发灰、低对比度
水底纹理模糊、边缘不清
颗粒噪声 / 悬浮物干扰
左亮右暗，非均匀光照

```

所以总体方法建议定为：

```text
基于非均匀光照校正、混合曝光修复与频域细节增强的水下图像增强方法
```

或者论文题目可以写成：

```text
面向人工光照水下场景的高光保护与频域增强图像复原方法
```

## 三、各论文在你方案里的作用

### 1. 黄奕程等：作为主框架

**论文：**《多支路差分卷积与频域增强相结合的水下图像增强方法》

这篇最适合作为你的水下增强主框架。它提出 CDEM、DBEM、FFM 等模块，用频域增强解决颜色失真和细节模糊，用多支路差分卷积提取边缘、纹理、高频细节，并用 SNRM 辅助抑制噪声。

对应解决：

```text
黄绿色偏色
纹理模糊
边缘不清
颗粒噪声
整体浑浊
```

建议你借鉴：

```text
CDEM：频域颜色和细节增强
DBEM：时空域 + 频域双支路增强
SDEM：ADC、RDC、CDC 多支路差分卷积
FFM：特征融合
边缘损失：增强纹理和边缘
```

------

### 2. 王悦等：作为注意力与对比学习补充

**论文：**《基于多尺度注意力和对比学习的水下图像增强》

这篇适合作为辅助参考。它采用编码—解码结构、多尺度通道像素注意力模块和对比学习，主要解决水下图像的颜色失真和细节模糊问题。

对应解决：

```text
色偏
模糊
细节不足
局部退化区域
```

建议你借鉴：

```text
像素注意力：让网络关注退化严重区域
通道注意力：增强重要颜色/纹理特征
对比学习：让增强图像接近清晰图，远离退化图
```

------

### 3. Ning 等：解决非均匀光照

**论文：** *Low-illumination underwater image enhancement based on non-uniform illumination correction and adaptive artifact elimination*

这篇专门针对低照度水下图像中的非均匀光照问题，提出非均匀光照校正、自适应伪影消除和多尺度融合，能够校正水下图像中非均匀低照明影响并减少光晕伪影。([Frontiers](https://www.frontiersin.org/journals/marine-science/articles/10.3389/fmars.2023.1249351/full?utm_source=chatgpt.com))

对应解决：

```text
左亮右暗
右侧暗区
局部照明不均
暗区增强后伪影
```

建议你借鉴：

```text
非均匀照明估计
暗区增强
亮区抑制
多尺度融合
伪影消除
```

------

### 4. RECNet：解决局部过亮 + 局部过暗共存

**论文：** *Region-Aware Exposure Consistency Network for Mixed Exposure Correction*

这篇针对 mixed exposure，即一张图像中同时存在过曝和欠曝区域的问题，提出区域感知曝光校正网络 RECNet，把不同曝光区域映射到统一曝光表示空间，并恢复局部信息。([arXiv](https://arxiv.org/abs/2402.18217?utm_source=chatgpt.com))

对应解决：

```text
局部光照太强
局部又太暗
同一张图曝光不一致
```

建议你借鉴：

```text
高光区域和暗区分区处理
区域感知曝光校正
曝光一致性约束
局部信息恢复
```

------

### 5. Cao 等：解决严重过曝校正

**论文：** *Over-exposure Correction via Exposure and Scene Information Disentanglement*

这篇针对过曝校正，核心是把曝光信息和场景内容信息解耦，再利用恢复网络生成校正结果。([CVF开放获取](https://openaccess.thecvf.com/content/ACCV2020/papers/Cao_Over-exposure_Correction_via_Exposure_and_Scene_Information_Disentanglement_ACCV_2020_paper.pdf?utm_source=chatgpt.com))

对应解决：

```text
严重过曝区域
亮区发白
过曝区域视觉不自然
```

建议你借鉴：

```text
曝光信息分离
场景内容信息保留
过曝区域修复
亮区与正常区域自然过渡
```

------

### 6. Eilertsen 等：解决高光区域细节丢失

**论文：** *HDR image reconstruction from a single exposure using deep CNNs*

这篇研究从单张普通图像重建 HDR 图像，目标是预测由于相机传感器饱和而丢失的亮部信息，例如高光和明亮区域。([arXiv](https://arxiv.org/abs/1710.07480?utm_source=chatgpt.com))

对应解决：

```text
高光区域细节丢失
白色亮斑内部无纹理
动态范围不足
```

建议你借鉴：

```text
饱和区域检测
HDR 细节预测
高光区域信息补全
亮区动态范围恢复
```

但要注意：如果区域已经完全纯白，这类方法只能**估计缺失信息**，不能保证真实恢复原始纹理。

## 四、你最终可以形成的完整方法

我建议你的方法分成 5 个核心模块：

```text
模块 1：高光检测与时间戳 Mask
作用：忽略时间戳，检测强反光和过曝区域

模块 2：非均匀光照校正模块
作用：解决左亮右暗、暗区细节不足

模块 3：混合曝光校正模块
作用：同时处理局部过曝和局部欠曝

模块 4：频域颜色与细节增强模块
作用：解决黄绿色偏色、浑浊、纹理弱

模块 5：差分卷积细节恢复与噪声抑制模块
作用：恢复边缘纹理，抑制颗粒噪声
```

对应结构：

```text
Input
 ↓
Mask / Highlight Detection
 ↓
Non-uniform Illumination Correction
 ↓
Mixed Exposure Correction
 ↓
Frequency-domain Color & Detail Enhancement
 ↓
Multi-branch Differential Convolution
 ↓
Noise Suppression + Feature Fusion
 ↓
Enhanced Output
```
