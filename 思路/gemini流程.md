### 阶段一：基于物理模型的离线数据合成 (Data Synthesis)

- **具体动作：**
  1. 收集开源的清晰图像（工业管道、水泥墙壁、水下地质图）。
  2. 编写 Python 脚本，利用单目深度估计网络（如 DepthAnything）生成对应的深度图。
  3. 结合物理光学退化公式 $I = Jt + A(1-t)$，人为随机注入环境光 $A$（黄绿色）、衰减系数 $\beta$、局部高斯光斑（高光过曝）和泊松噪声（悬浮颗粒）。
  4. 生成海量 `(浑浊图 I, 真实清晰图 J, 真实透射率 t, 真实环境光 A)` 的四元组数据。
- **解决的问题：** 打破了深度学习最致命的“零成对数据”限制。为你提供带有完美 Ground Truth（真实标签）的训练集。
- **可借鉴论文：**
  - *AOD-Net (ICCV 2017)*: 证明了基于 Koschmieder 公式合成的数据可以训练出泛化能力极强的去雾网络。
  - *UWCNN (Pattern Recognition 2020)*: 详细演示了如何根据水质类型合成水下退化图像集。

### 阶段二：双分支物理引导网络搭建 (Network Architecture)

- **具体动作：** 使用 PyTorch 搭建网络，强制将其分为三个子模块：
  1. **参数估计子网：** 输入浑浊图，输出环境光 $A$（1x1向量）和 透射率 $t$（单通道特征图）。
  2. **纯数学物理层：** 无参数层，直接执行反演公式计算出粗糙清晰图 $J_{rough} = \frac{I - A}{t} + A$。
  3. **残差精炼子网：** 引入轻量级去噪 Block，输入 $J_{rough}$ 和原图，输出去除颗粒和伪影后的最终清晰图。
- **解决的问题：** 彻底抛弃了“黑盒瞎猜”模式，解决了模型缺乏可解释性的问题。将整体偏色（A）、发灰（t）与局部颗粒噪声（精炼网络）进行了解耦处理。
- **可借鉴论文：**
  - *UColor (TIP 2021)*: 论证了将透射率估计和颜色特征融合在网络中的有效性。
  - *Deep Retinex Decomposition (BMVC 2018)*: 提供了一种经典的“解耦（光照与反射率）然后再组合”的网络设计思路。

### 阶段三：联合监督预训练 (Supervised Pre-training)

- **具体动作：**
  1. 将阶段一合成的四元组数据喂给阶段二搭建好的网络。
  2. 设计多级损失函数（Loss）：不仅用 $L_1$ 损失对比最终输出和真实清晰图，还直接计算网络预测的 $A$、$t$ 与合成脚本生成的真实 $A$、$t$ 之间的 MSE（均方误差）。
  3. 用 AdamW 优化器进行训练，直至网络在合成数据上收敛。
- **解决的问题：** 赋予网络基础的“去偏色、去雾、去噪”能力。由于我们直接监督了物理参数，网络收敛速度会比普通 U-Net 快好几倍，极大地节省算力。
- **可借鉴论文：**
  - *Perceptual Losses for Real-Time Style Transfer and Super-Resolution (ECCV 2016)*: 解释了为什么要引入感知损失（VGG Loss）来恢复水底纹理的边缘锐度。

### 阶段四：真实场景无监督微调 (UDA Fine-tuning)

- **具体动作：**
  1. 冻结（Freeze）精炼子网的参数，只保持“参数估计子网”可训练。
  2. 把你手里那些**只有浑浊、没有清晰配对的真实管道图像**喂给网络。
  3. 使用**无参考损失函数**（如要求输出图像的三通道均值一致以去除偏色，要求透射率图空间平滑以避免伪影）对模型进行少量步骤（约 500-1000 步）的微调。
- **解决的问题：** 解决虚实域偏差（Sim-to-Real Domain Gap）。让在“假数据”上训练好的网络，完美适应你真实管道中的独有光场环境和特殊颗粒分布。
- **可借鉴论文：**
  - *Zero-DCE (CVPR 2020)*: 详细介绍了如何在没有清晰图像的情况下，利用颜色恒常性损失、空间平滑损失等无参考指标来训练网络。
