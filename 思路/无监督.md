------

# 一、传统图像处理方法（最适合论文和工程）

这类方法最像：

```text
OpenCV + 数学模型
```

优点：

- 可解释
- 可复现
- 好写论文
- STM32 / Jetson / ROS 都容易部署

缺点：

- 极端反光恢复能力有限
- 效果上限低于生成式 AI

------

## 1. Retinex 系列（非常推荐）

用于：

- 去灰雾
- 提升局部对比度
- 改善光照不均

经典：

### MSRCR

R_i(x,y)=C_i\left[\log I_i(x,y)-\log(F(x,y)*I_i(x,y))\right]

你这张图非常适合。

GitHub：

[RetinexNet GitHub](https://github.com/weichen582/RetinexNet?utm_source=chatgpt.com)

------

## 2. CLAHE（局部对比度增强）

用于：

- 提高清晰度
- 增强石子纹理
- 提升边缘

OpenCV 直接有：

```python
clahe = cv2.createCLAHE()
```

你这张图用了会明显提升底部纹理。

------

## 3. DCP 去雾（暗通道）

经典去雾：

J(x)=\frac{I(x)-A}{t(x)}+A

用于：

- 去浑浊
- 提高清晰度

水下版很多论文都在用。

GitHub：

[Dark Channel Prior Dehazing](https://github.com/ForeverPs/haze-removal-using-dark-channel-prior-and-guided-filter?utm_source=chatgpt.com)

------

# 二、深度学习水下增强（最接近我刚刚效果）

FUnIE-GAN

# 三、生成式修复（最接近我刚刚那个“AI 修图”）

你真正想问的，其实是：

```text
有没有开源版本能像 ChatGPT 一样：
上传图 → 写要求 → 自动修图
```

答案是：

有，而且越来越多。

------

## 1. Stable Diffusion + ControlNet

这个已经非常接近我刚刚的方法。

流程：

```text
原图
+
文字提示：
去偏色
增强纹理
去反光
恢复细节
↓
生成增强图
```

这已经是“开源版 AI 图像编辑”。

GitHub：

[Stable Diffusion WebUI](https://github.com/AUTOMATIC1111/stable-diffusion-webui?utm_source=chatgpt.com)

[ControlNet GitHub](https://github.com/lllyasviel/ControlNet?utm_source=chatgpt.com)

------

## 2. InstructPix2Pix（非常像我刚刚的操作）

这个你一定要看。

它就是：

```text
输入图片
+
文字要求
↓
输出修改后的图片
```

例如：

```text
remove haze
enhance underwater texture
reduce glare
```

这和我刚刚干的事情非常接近。

GitHub：

[InstructPix2Pix GitHub](https://github.com/timothybrooks/instruct-pix2pix?utm_source=chatgpt.com)

论文：

InstructPix2Pix
